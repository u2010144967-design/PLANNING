Récapitulatif du mois

Synchronisation

Renseignez le projet Supabase pour que vous et votre collègue travailliez sur la même répartition. Sans ces informations, l'application fonctionne quand même : les données restent dans cet ordinateur.

Table à créer une seule fois dans l'éditeur SQL de Supabase :

create table if not exists repartition_aout (
  id text primary key,
  data jsonb not null,
  auteur text,
  updated_at timestamptz not null default now()
);
alter table repartition_aout enable row level security;
create policy "acces_public" on repartition_aout
  for all using (true) with check (true);

Mode d'emploi

Cette application contient la répartition des tournées déjà préparée, semaine par semaine, du 3 août au 6 septembre. Il n'y a rien à décider : tout est fait.

Chaque jour de livraison

  1. Ouvrez l'onglet Distribuer les bons. La semaine en cours est sélectionnée automatiquement (repère « aujourd'hui »).
  2. Vous voyez 4 colonnes, une par chauffeur, avec la liste des clients qui lui reviennent.
  3. Remettez à chaque chauffeur les bons de livraison de sa colonne, et cochez chaque client au fur et à mesure : la ligne se barre et le compteur avance.
  4. En cas de doute, imprimez : Imprimer les 4 fiches de la semaine sort une feuille A4 par chauffeur, à agrafer avec ses bons.

Points d'attention

  • Un bandeau rouge signale les clients sans chauffeur : ce sont les seuls cas à trancher, appelez le responsable.
  • Les clients marqués « à confirmer » (semaine du 31/08) ne sont pas encore certains : vérifiez auprès du client avant de sortir le bon.
  • Le cadre Consignes en bas reprend les instructions laissées pour la semaine.

Si vous devez modifier quelque chose

L'onglet Préparer la répartition permet de changer l'affectation d'un client d'un clic. S'il est verrouillé, un code est demandé — demandez-le au responsable. Tout est enregistré automatiquement, et si la synchronisation est active, l'autre poste voit le changement en moins d'une minute.

Installer sur l'ordinateur

Sur Chrome ou Edge, ouvrez le menu ⋮ en haut à droite, puis Installer « Répartition des tournées » (ou l'icône d'installation dans la barre d'adresse). L'application s'ouvrira ensuite dans sa propre fenêtre, avec un raccourci sur le bureau, et fonctionnera même sans connexion.

Sur téléphone : menu du navigateur → Ajouter à l'écran d'accueil.

Si le bouton d'installation n'apparaît pas, c'est que la page a été ouverte depuis un fichier local. Ouvrez-la depuis son adresse web : l'installation devient possible.
