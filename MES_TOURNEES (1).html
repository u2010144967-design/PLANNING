Renseignez le projet Supabase pour que vous et votre collègue travailliez sur la même répartition. Sans ces informations, l'application fonctionne quand même : les données restent sur cet ordinateur.

Codes d'accès

Le code 2026 ouvre l'application avec l'accès complet, et il fonctionne en permanence. Vous pouvez en plus donner un code à votre collègue : avec celui-là, il pourra remettre les bons et cocher, mais pas modifier la répartition.

Rassurez-vous : le code 2026 fonctionnera toujours, même si vous oubliez celui que vous mettez ici.

Mode d'emploi

Cette application contient la répartition des tournées déjà préparée, semaine par semaine, du 3 août au 6 septembre. Il n'y a rien à décider : tout est fait.

À l'ouverture

Tapez votre code d'accès. Avec le code distribution, vous arrivez directement sur l'écran de remise des bons — c'est le seul dont vous avez besoin. Le bouton Déconnexion, en haut à droite, referme l'accès quand vous quittez le poste.

Chaque jour de livraison

  1. La semaine en cours est sélectionnée automatiquement (repère « aujourd'hui »).
  2. Vous voyez 4 colonnes, une par chauffeur, avec la liste des clients qui lui reviennent.
  3. Remettez à chaque chauffeur les bons de livraison de sa colonne, et cochez chaque client au fur et à mesure : la ligne se barre et le compteur avance.
  4. Au besoin : Imprimer les 4 fiches de la semaine sort une feuille A4 par chauffeur, à agrafer avec ses bons.

Points d'attention

  • Un bandeau rouge signale les clients sans chauffeur : appelez le responsable.
  • Les clients marqués « à confirmer » (semaine du 31/08) ne sont pas certains : vérifiez auprès du client avant de sortir le bon.
  • Un client marqué 👥 est livré par deux chauffeurs ensemble : il apparaît dans les deux colonnes.
  • Le cadre Consignes en bas reprend les instructions laissées pour la semaine.

Installer sur l'ordinateur

Sur Chrome ou Edge, ouvrez le menu ⋮ en haut à droite, puis Installer « Répartition des tournées » (ou l'icône d'installation dans la barre d'adresse). L'application s'ouvre ensuite dans sa propre fenêtre, avec un raccourci sur le bureau, et fonctionne même sans connexion.

Sur téléphone : menu du navigateur → Ajouter à l'écran d'accueil.

Si le bouton n'apparaît pas, c'est que la page a été ouverte depuis un fichier local. Ouvrez-la depuis son adresse web : l'installation devient possible.
